## 🧠 Funcionalidades

- 🎯 Seleção de dificuldade (Fácil, Médio, Difícil)
- 📚 Categorias diferentes por dificuldade
- ⌨️ Navegação por teclado (setas + Enter)
- 🔤 Suporte a letras acentuadas (á, ç, õ, etc.)
- 🧩 Suporte a palavras com espaço
- 🧍 Desenho da forca atualizado conforme os erros
- 💡 Possibilidade de chutar a palavra inteira (TAB)

---

## 🎮 Como jogar

1. Escolha a dificuldade usando **←→**
2. Escolha a categoria usando **↑↓**
3. Pressione **ENTER** para começar
4. Digite letras para tentar adivinhar a palavra
5. Pressione **TAB** para tentar adivinhar a palavra inteira
